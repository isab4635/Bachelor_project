# Bachelor_project

This repository is for the bachelor project "Treating rheumatoid arthritis — trajectories, correlations and predictions based on a large nationwide dataset" running during spring 2026.

The code found here is copies of the scripts on the remote server, which is not synced to GitHub. 

There are two folders: In_progress and Scripts. Like the name may indicate the In_progress folder are for scripts where some adjustment is still needed before they are used. Scripts contain those working, though these may still be subject to change and improvements.
